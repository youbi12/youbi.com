Installation

In order to be up and playing with this plugin please follow this tutorial on how to install Couponis Theme.

Inside of the Couponis_WP/01_Couponis_Theme folder you will find the theme zip caller couponis.zip. For those who are familiar with the theme installation is the same as for any other theme. For those who do not know how to install the theme, please follow this short installation tutorial.

 

Installation Procedure

The first thing you need to do when you want to install a new WordPress theme is to login to your site admin page. Once there, go to Appearance -> Themes.

Here, you will see all the themes you have currently installed in your application. To add another one, simply click on the Install Themes tab.

To installl Couponis theme click on the Upload link at the top of the page.

Now, click the Choose File button, select the archive of your theme from your local computer and press the Install Now button.

In few seconds, WordPress will upload and extract the theme archive for you. All you need to do next is to press the Activate link under the message for successful theme installation.

After the theme has been installed, there will be notification about to isntall required plugins for the theme. Click on the link Begin Installation and on the next page select all plugins and select bulk action Install

After that click on Return To The Required Plugins Installer and again select all the plugins and this time select bulk action Activate.

Demo Content

In order to import demo you need to follow these steps.

1. Go to Appearance->Import Demo Data and click on Import Demo Data and wait for the process to finish. If you receive errors 500 or 404 then make sure that max_execution_time on your server configuration (php.ini) is set high enough ( it would be best to set it to 0 - zero for unlimited time )

2. Go to Settings->Permalinks and set permalinks to Post Name and click Save Changes

Coupon Post Type

Coupon options are built using native WordPress way so you will already be familiar with everything. Available coupon options are:

Coupon Description

Coupon Category

Coupon Store

Coupon Image - image is displayed on coupon single page while on the coupon listings you can select to have either this image or store logo ( you set this in theme options )

Coupon Type - depending on the selected value of this field group of suboptions will appear

- Online Code - here you input code which wil be presented for the users on frontend

- In Store Code - here you will be prompted to add image of the coupon which user will print out and take to the store

- Online Sale - here you will put link to the online sale and user will be forwarded to this link

Is Exclusive - if the coupon is only available on your site check this field to make users aware of it

Used - here you will see number of opening the coupon

Affiliate Link - when link is present here, and user click on action button, he will be moved to new tab with code while in the current tab link will be opened

 

When user submits coupon on frontend you will receive the mail and coupon will not be published untill you review it.

WP All Import

Theme fully supports WP All Import and it takes it even fuirther since it allows you to import everyhing using free version of the plugin. This tutrial will assume that you've read documetation of the WP All Imnport plugin and that you are familiar with it.

1. Once you've installed WP All Import make sure that plugin Couponis Import is installed and activated

2. Once you've uploaded your XML/CSV you are presented with the screen of configuring data. for import. You will notice new toggle called Couponis Coupo and that's where the extra fields are.

3. First field is 'Coupon Type' where you can select what type of the coupons are you importing.

If all of them are Online Code then you can just select that one from the radio select choices ( similar for the other two ). If you are importing coupons with mixed types you will select last option 'Set With xPath' and then you will drag tag, which holds this information, from the XML file to the input field. Values which theme requires are:

  • Online Code - needs value 1
  • In Store Code - needs value 2
  • Online Code - needs value 3

If your selected tag holds values 1, 2, or 3 and then you donot need to map the values. If this is not the case then ou need to click on Field Options -> Mapping which is located next to the input field. Now you will be presented with the window in which you need to input values from your tag which will be translated. For example if your tag which describes coiupon type holds values: o-code, in-code, online ( values for three different types ) then you type in those values in the fields 'In Your File' next to the appropriate value to which it will be translated. In our example you will type o-code ( Online Code ) next to the value of 1. After that you just click Save Rules

Next field is Coupon Expire where you just locate field with that value and drag it to input field ( if you do not see the value of this or any other field you browse through XML file using arrows and locate example of it )

Then you have Coupon Data set of options where you will find example of In Store Coupon in your XML ( if you have it or otherwise leave this empty ) and drag tag which holds link to the image to Printable Coupon Image option do the same for the fields Coupon Code Coupon URL and Coupon Affiliate URL

If you are using free version of the WP All Import plugin then in this section you will see option Coupon featured Image where you will drag and drop tag which holds value for the coupon image otherwise this will be removed and default option of paid version will be used instead.

Coupon Taxonomies Coupon Category and Coupon Store are handled by default WP All Import options for taxonomies.

Couponis Import assumes that each coupon is assigned to only one store.  In order to import store URL and logo ou will drag and drop tags of each value to inputs of Store URL and Store Logo and plugin will then connect it to appropriate store assigne in Taxonomy section.

Of course you can always change all of these manually after the import

 

 

FAQ

1. API Key added but map is not loaded or search by places does not show any suggestions

You have issue with your API. Open browser console F12 switch to Console tab and read errors which google gives you and fix them ( ignore warnings )

 

2. Site is broken after theme activation

PHP version on your server is outdated. Contact hosting company so they can update it per WordPress requirements  https://wordpress.org/about/requirements/

 

3. How to translate the theme?

a. Download .pot from languages folder
b. Open it in poEdit program and translate strings
c. Save file with your language code as file name
d. Upload generated po and mo file back to languages folder
e. Change language in Settings->General.

 

4. How to update the theme?

a. Download All Files & Documentation to get changelog
b. If you did not edited theme files delete current version and install new one or install plugin http://envato.github.io/wp-envato-market/ for easier update.
c. If you did edited then move modifications to new files and install the theme
d. If you have edited theme files via child theme switch to parent theme and update it then update child theme files which are changed in recent update ( list of changed files is in changelog )

 

5. Demo content not fully imported

If you did not saw All Done message then your server configuration for max script execution time is set to low. Contact your hosting company and tell them to increase it or set to 0 ( zero ) for unlimited

 

6. Why images after import are grey ones?

Images from demo are not included in the demo content, in place of them, placeholder image is being used ( same behavior like demo images ) in order not to violate image license.
 

 

7. Coupons in tabs have white background?

Add coupon-tabs class here https://prnt.sc/233jux3

 

8. Elementor tabs like in demo?

Add coupon-tabs class here https://prnt.sc/26rcf7b

 

9. Switch from KingComposer to Elementor

If you have imported demo content then in Appearance -> Import Demo Data you will see new button Import Elementor Data which will only import demo pages built for Elementor. Go to Pages pick which Home page you wish to use and start editing

If you are importing demo for the first time it wil autoimatically import for Elementor
